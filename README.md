# Mini Projet - Calculateur de Proteines

Application web pour calculer les besoins en proteines selon le poids et les objectifs de l'utilisateur.

## Technologies

- React 19.2.0
- TypeScript 5.9.3
- Vite 8.0.0-beta.13
- Node.js 23.3.0
- npm 10.9.0

## Installation et lancement

```bash
npm install
npm run dev
```
